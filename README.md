**Cài Git vào Linux**

Nếu bạn đang sử dụng hệ điều hành Ubuntu/Debian thì có thể sử dụng lệnh sau để cài Git:

`$ sudo apt-get install git`

Hoặc lệnh sau để cài trên **CentOS/Fedora/RHEL**:

`$ yum install git`
